### **E-Commerce MERN Stack Project for Farmers**  

This project is an **E-Commerce platform** built using the **MERN stack (MongoDB, Express.js, React, and Node.js)**. It is specifically designed to help **farmers rent and purchase essential resources** needed for farming. ## **How to Run This Project Locally**  

Follow these steps to set up and run the project on your local machine:  

### **1. Clone the Repository**
git clone **repository-url** <br>

cd **project-folder**

### **2. Install Dependencies**
npm install

### **3. Start the Development Server**

npm run dev

### **4. Access the Project**
Open your browser and visit: http://localhost:5173
